# DayZ External

External overlay for DayZ with ESP and menu.

## Build

Requirements:
- Visual Studio 2019/2022
- DirectX SDK

```
Open DayZ Steam.sln
Select Release x64
Build (F7)
```

Or run `build.bat`

## Usage

1. Start DayZ
2. Run Steam.exe as admin
3. Press INSERT for menu

## Features

- Player/Zombie/Loot ESP
- Aimbot
- Config system
- Performance optimizations

See BUILD.md for details.
